# 🏨 Hotel Quinta Dalam - Proyecto Web

Este es un proyecto de desarrollo web multipágina creado para la asignatura de Programación Web. El sitio web representa a un hotel boutique temático inspirado en la riqueza cultural de los **Pueblos Mágicos de Michoacán**.

## 🚀 Características
- **Arquitectura de 13 Habitaciones:** Cada habitación del catálogo está dedicada a un municipio o artesanía representativa (Pátzcuaro, Tzintzuntzan, etc.).
- **Diseño Responsivo:** Implementado con **CSS Grid** y **Flexbox** para una visualización óptima en dispositivos móviles y de escritorio.
- **Interactividad:** Uso de **JavaScript** nativo para efectos dinámicos en el catálogo de imágenes.
- **Control de Versiones:** Gestionado íntegramente con Git y GitHub.

## 📂 Estructura del Proyecto

La organización del código sigue los estándares de limpieza y separación de intereses (SoC):

```text
Hotel-Quinta-Dalam/
├── css/                # Hojas de estilo estructuradas
│   └── styles.css      # Reglas de diseño, Grid y Media Queries
├── js/                 # Lógica de programación
│   └── main.js         # Scripts para efectos y validaciones
├── img/                # Activos multimedia
│   ├── habitaciones/   # Fotos del catálogo de los 13 municipios
│   └── logo.png        # Identidad visual
├── index.html          # Punto de entrada (Misión, Visión e Inicio)
├── catalogo.html       # Visualización dinámica de habitaciones
├── nosotros.html       # Historia y valores del hotel
├── contacto.html       # Formulario y ubicación (Google Maps)
└── README.md           # Documentación del repositorio
